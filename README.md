# Plataforma de Evaluación Socioemocional

Aplicación web responsive para evaluación socioemocional de estudiantes (1° básico a 4° medio), basada en las 5 competencias emocionales de Rafael Bisquerra.

## Funcionalidades

- Encuesta Likert (1-5) adaptativa por ciclo vital.
- Preguntas abiertas con análisis semántico básico por dimensión.
- Cálculo de promedio por dimensión, promedio general, nivel interpretativo y desviación estándar.
- Reportes individuales y grupales con visualizaciones (barras y radar).
- Recomendaciones de mejora automáticas alineadas a educación emocional.
- Exportación CSV, Excel, PDF y gráficos PNG.
- Panel administrativo con filtros por curso y fecha.
- Anonimización opcional al guardar registros.
- Índice global de bienestar socioemocional y alertas por puntaje crítico.

## Uso rápido

1. Abrir `index.html` en un navegador moderno o ejecutar un servidor estático.
2. Completar formulario y enviar.
3. Revisar resultados individuales y panel grupal.
4. Usar exportaciones según necesidad.

## Nota

Esta versión almacena datos localmente en `localStorage`. Para producción, integrar backend seguro, autenticación y cifrado.
